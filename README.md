# 📝 Personal Blog Application

A **full-stack blog application** built for personal use, where I can create and publish blogs, and users can read articles to gain knowledge on different topics.  
The platform is designed with a **clean UI and user-friendly experience**, allowing readers to easily explore content.

---

## 🚀 Features

- ✍️ Create and submit blogs with **title, content, and author details**  
- 📖 Users can browse and read blogs in a **simple, organized layout**  
- 📱 **Responsive design** for seamless use across devices  
- ⚡ Backend API for **storing and fetching blog data**  
- 🔒 Secure and scalable structure for **future enhancements**  

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Version Control:** Git & GitHub
